Sobre o Projeto de Gestão de Matrículas

O projeto de gestão de matrículas de alunos foi desenvolvido para facilitar o gerenciamento de matrículas em uma escola ou instituição de ensino. Ele oferece recursos para cadastrar alunos, matrículas, buscar informações relevantes e muito mais.

O projeto utiliza as seguintes tecnologias:

  • Feito por João Antonio Da Silva
  • Back-end: Java com Spring Boot
  • Front-end: HTML, CSS e JavaScript
  • Banco de Dados: PostgreSQL
