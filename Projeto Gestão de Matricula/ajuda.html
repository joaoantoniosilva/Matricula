Ajuda - Gestão de Matrícula

Esta é a página de ajuda do seu site de gestão de matrícula. Aqui estão algumas dicas para usar o sistema:

  • Passo 1: Faça login na sua conta.
  • Passo 2: Acesse a página de matrículas.
  • Passo 3: Selecione a turma desejada.
  • Passo 4: Clique no botão "Matricular Aluno" para adicionar um aluno.
  • Passo 5: Preencha os dados do aluno e confirme a matrícula.

Se você tiver algum problema ou dúvida, entre em contato conosco para obter suporte.
